# Tools Documentation - n8n Automation Agent

El n8n Automation Agent tiene acceso a herramientas de diagnóstico e integración:

1.  **API de n8n:**
    *   Listar flujos de trabajo (workflows).
    *   Listar y descargar datos de ejecuciones históricas (`/api/v1/executions`).
    *   Activar/desactivar workflows.
2.  **Entorno de Depuración JavaScript:**
    *   Ejecutar y validar lógica de transformación de datos localmente antes de proponerla.
