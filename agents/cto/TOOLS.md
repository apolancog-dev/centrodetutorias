# Tools Documentation - CTO Agent

El CTO Agent tiene acceso a herramientas de nivel de sistema y repositorio:

1.  **Git / Control de Versiones:**
    *   Comandos Git para clonar, checkout, commit, push, branch y pull.
2.  **Edición de Código:**
    *   Comandos bash para leer y modificar código en la estructura de la landing page.
3.  **Comprobación Técnica:**
    *   Acceso a utilidades de formateo (Linter, Prettier) o empaquetadores locales instalados en el proyecto.
