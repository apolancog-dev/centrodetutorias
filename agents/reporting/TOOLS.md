# Tools Documentation - Reporting Agent

El Reporting Agent tiene acceso a herramientas de base de datos y procesamiento de reportes:

1.  **API REST de Supabase / Base de datos:**
    *   Hacer consultas SQL o REST para contar registros, leads e interacciones en Supabase.
2.  **API de Paperclip:**
    *   Consultar costos acumulados por corrida y agente (`/api/runs/cost`).
3.  **Procesamiento Markdown:**
    *   Capacidad de formatear textos complejos y exportar informes.
