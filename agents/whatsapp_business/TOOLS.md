# Tools Documentation - WhatsApp Business Agent

El WhatsApp Business Agent tiene acceso a herramientas de comunicación e integración:

1.  **Integración de CRM (GoHighLevel / GHL API):**
    *   Crear y actualizar contactos.
    *   Mover oportunidades de etapa en el Pipeline (ej. de "Nuevo Lead" a "Diagnóstico Agendado").
2.  **Mensajería (WhatsApp Business API / n8n Webhook):**
    *   Enviar mensajes de texto enriquecidos con plantillas aprobadas.
    *   Enviar enlaces interactivos de agendamiento (Calendly o similar).
3.  **Consulta de Calendario:**
    *   Consultar disponibilidad de horarios de diagnóstico.
